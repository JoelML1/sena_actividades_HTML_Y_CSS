Glosario de TI

N - Node.js

Entorno de ejecución para JavaScript que permite construir aplicaciones del lado del servidor.

O - Open Source

Software cuyo código fuente está disponible para ser modificado y distribuido libremente.

P - Python

Lenguaje de programación versátil y fácil de aprender, utilizado en desarrollo web, ciencia de datos y más.

Q - Query

Consulta utilizada para recuperar información de una base de datos.

R - REST

Arquitectura para servicios web que utiliza métodos HTTP para interactuar con recursos.

S - SQL

Lenguaje de consulta estructurado utilizado para gestionar bases de datos relacionales.

T - TCP/IP

Conjunto de protocolos que permiten la comunicación entre computadoras en una red.

U - UX

Experiencia de usuario, enfocada en cómo las personas interactúan con un producto o servicio.

V - Virtualización

Tecnología que permite ejecutar múltiples sistemas operativos en un solo hardware físico.

W - Wi-Fi

Tecnología que permite la conexión inalámbrica a redes de internet.

X - XML

Lenguaje de marcado utilizado para almacenar y transportar datos.

Y - YAML

Lenguaje de serialización de datos legible por humanos, utilizado en configuración de aplicaciones.

Z - Zero Trust

Modelo de seguridad que asume que ninguna entidad, interna o externa, es confiable por defecto.
