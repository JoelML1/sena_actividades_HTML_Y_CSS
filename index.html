Glosario de TI

A - Algoritmo

Conjunto de instrucciones secuenciales que permiten resolver un problema o realizar una tarea específica.

B - Big data

Conjunto de datos masivos y complejos que requieren herramientas especiales para su almacenamiento, procesamiento y análisis.

C - Cloud computing

Entrega de servicios informáticos (servidores, almacenamiento, bases de datos, etc.) a través de internet.

D - DNS (Domain Name System)

Sistema que traduce nombres de dominio legibles por humanos (como google.com) en direcciones IP.

E - Encriptación

Proceso de codificar información para protegerla de accesos no autorizados.

F - Firewall

Dispositivo o software que filtra el tráfico de red, bloqueando accesos no deseados.

G - GPU

Procesador especializado en cálculos gráficos y paralelos, común en videojuegos y aprendizaje automático.

H - HTML (HyperText Markup Language)

Lenguaje estándar para crear páginas web.

I - IoT (Internet of Things)

Red de objetos físicos conectados a internet que recopilan e intercambian datos.

J - JavaScript

Lenguaje de programación popular para aplicaciones empresariales, móviles y web.

K - Kernel

Núcleo del sistema operativo que gestiona el hardware y coordina el software.

L - Linux

Sistema operativo de código abierto basado en Unix, conocido por su estabilidad y seguridad.

M - Malware

Software malicioso diseñado para dañar, interrumpir o acceder a sistemas informáticos sin autorización.
